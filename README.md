## Instruções do Repositório

Esse repositório será destinado a estudos de C# com foco no Backend e Cloud, ele servirá como diário de estudos, onde registrarei tópicos e conteúdos que achar interessante ou que agreguem algum conhecimento ou prática a ser testada implementada. 


O objetivo é reunir o progresso de estudos nessa área para que possa ser resgatado em ocasiões futuras caso necessário.
